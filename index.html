CORONEWS-19 API

CORONEWS-19 is a restful API built on top of python flask which delivers the latest news globally on COVID-19 and also the stats of COVID-19 in India. This API is made for developers by developer. This API response data in form of JSON. You can use JQUERY or Python to get data out of this API and then you can use a for loop to get all the news article.
